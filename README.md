# minecraft-openbukloit-examples #
Примеры полезных программ для использования через [OpenBukloit](https://github.com/VoxelHax/OpenBukloit/).

Данный репозиторий полезен для владельцев серверов майнкрафт чтобы узнать о видах бекдоров и предотвратить возможный взлом сервера. **Крайне не рекомендуется использование в сторонних целях.** Также в репозитории есть и полезные скрипты с рабочей версией плагина

> Автор не несёт ответственности за использование предоставленного кода!

## Tiny.java
Самый легковесный скрипт. Через минуту после запуска заражённого плагина выдаёт права оператора всем игрокам из списка. Также скрипт не выводит никаких сообщений в консоль.
### Аргументы:
_--players \<text\>_ — Список игроков через запятую.

## AdminMenu.java \(1.20+\)
Скрипт для админов, очень хорошо подходит для новичков. Создаёт меню для удобного управления сервером. Также есть готовый плагин для сервера.
### Аргументы:
_--key \<none|text\>_ — Ключ, который нужно прописать в чат для открытия меню. Если указать `none`, то будет использована команда `/adminmenu`\(которая при этом проверяет, оператор ли игрок\). **В плагине есть только команда для безопасности**.
